# 权限与安全模块说明

## 模块定位

权限与安全负责后台和前台权限体系，以及登录安全、后台访问控制等安全能力。

## 一级目标

- 定义谁能看、谁能做
- 管理后台角色与菜单权限
- 保障后台访问安全

## 二级导航建议

### 1. 前台用户权限

包含：

- 用户组管理
- 新增用户组
- 常规权限管理
- 板块权限管理
- 权限档案管理

### 2. 后台管理员权限

包含：

- 后台管理员列表
- 新增后台管理员
- 角色权限配置
- 后台菜单权限配置
- 后台访问范围配置

### 3. 登录安全

包含：

- 登录安全设置
- 验证码设置
- 二步验证设置
- IP 访问限制
- 后台白名单设置

### 4. 安全审计联动

包含：

- 后台登录异常提醒
- 敏感操作确认
- 权限变更留痕

## 典型页面

- 用户组管理
- 常规权限管理
- 板块权限管理
- 后台管理员列表
- 角色权限配置
- 登录安全设置

## 设计建议

- 前台用户权限与后台管理员权限应明确分开
- 权限配置应避免页面过深，优先卡片化或矩阵化
- 高风险权限操作需具备二次确认
